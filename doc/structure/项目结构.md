# 项目结构

> 文档位置：`doc/structure/项目结构.md`  
> 对应仓库：`my-nl2sql`

## 目录总览

```
my-nl2sql/
├── doc/                          # 项目文档
│   ├── plan/                     # 架构设计方案
│   │   └── NL2SQL 系统设计.md
│   ├── structure/                # 项目结构说明
│   │   └── 项目结构.md
│   └── startup/                  # 启动指南
│       └── 项目启动.md
├── apps/                         # 前端应用（npm workspaces）
│   ├── web/                      # 用户问答界面 (Next.js, :4000)
│   └── admin/                    # 管理后台 (Next.js, :4001)
├── packages/                     # 共享包
│   └── graphql-schema/           # GraphQL 共享类型定义
├── backend/                      # Python 后端
│   ├── api/                      # FastAPI + Strawberry GraphQL
│   ├── graph/                    # LangGraph 工作流
│   ├── rag/                      # Qdrant 混合检索与索引
│   ├── sql/                      # FK 图谱、SQL 校验与执行
│   ├── cache/                    # LLM 混合缓存
│   ├── llm/                      # LLM 调用封装（含缓存）
│   ├── workers/                  # Celery 异步任务
│   ├── db/                       # ORM 模型、Prompt 管理、数据库会话
│   ├── scripts/                  # 种子数据等脚本
│   └── tests/                    # 单元测试
├── alembic/                      # 数据库迁移
│   └── versions/
├── docker-compose.yml            # MySQL + Qdrant + Redis
├── alembic.ini
├── pyproject.toml                # Python 依赖与项目配置
├── package.json                  # npm workspaces 根配置
├── Makefile                      # 常用命令快捷方式
├── .env.example                  # 环境变量模板
└── README.md                     # 项目简介
```

## 后端模块说明

| 路径 | 职责 |
|------|------|
| `backend/api/main.py` | FastAPI 应用入口，挂载 GraphQL 路由 |
| `backend/api/schema.py` | Strawberry GraphQL Query / Mutation / Subscription |
| `backend/api/session_manager.py` | 问答会话管理与流式事件队列 |
| `backend/graph/workflow.py` | LangGraph 9 节点 NL2SQL 工作流 |
| `backend/rag/retriever.py` | Qdrant BM25 + 向量 + RRF 混合检索 |
| `backend/rag/indexer.py` | 元数据 / 模板 / 术语 / FK 索引管道 |
| `backend/sql/schema.py` | networkx FK 图谱、JOIN 路径、sqlglot 校验 |
| `backend/cache/llm_cache.py` | 精确 + 语义混合 LLM 缓存 |
| `backend/llm/service.py` | 带缓存的 LLM 调用服务 |
| `backend/workers/tasks.py` | Celery：RAG chunk 异步评分与告警 |
| `backend/db/models.py` | SQLAlchemy ORM 全量模型 |
| `backend/db/prompts.py` | 9 角色 Prompt 加载与版本管理 |
| `backend/config.py` | 环境变量与系统配置 |
| `backend/scripts/seed.py` | 开发环境种子数据 |

## 前端应用说明

### 用户端 `apps/web`

| 路径 | 说明 |
|------|------|
| `app/page.tsx` | 问答主界面：输入、深度思考、执行模式、流式展示 |
| `app/providers.tsx` | Apollo Client Provider |
| `lib/apollo.ts` | HTTP + WebSocket 双链路 Apollo 配置 |

### 管理后台 `apps/admin`

| 路径 | 说明 |
|------|------|
| `app/(admin)/page.tsx` | 系统概览仪表盘 |
| `app/(admin)/prompts/page.tsx` | Prompt 版本化 CRUD |
| `app/(admin)/metadata/page.tsx` | 数据源元数据与 RAG 索引 |
| `app/(admin)/cache/page.tsx` | LLM 缓存监控 |
| `app/(admin)/alerts/page.tsx` | RAG 低分告警列表 |

## 基础设施服务

| 服务 | 端口 | 用途 |
|------|------|------|
| MySQL 8.0 | 3307 | 元数据、Prompt、缓存、告警、会话 |
| Qdrant | 6333 / 6334 | 向量检索（BM25 + dense + RRF） |
| Redis | 6379 | Celery broker / result backend |
| FastAPI | 8000 | GraphQL API |
| Web 前端 | 4000 | 用户问答界面 |
| Admin 前端 | 4001 | 管理后台 |
